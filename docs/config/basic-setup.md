# Basic Setup

Configure the basic settings for your project.

## Environment Variables

```env
API_KEY=your_api_key
DEBUG=false
ENVIRONMENT=production
```

## Project Configuration

```json
{
  "name": "my-project",
  "version": "1.0.0",
  "settings": {
    "theme": "dark",
    "language": "en"
  }
}
```
